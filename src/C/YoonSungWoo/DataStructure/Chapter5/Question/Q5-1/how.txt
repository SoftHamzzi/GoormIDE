1. 직원정보를 등록할 수 있다. 직원 정보는 사번과 이름으로 구성이 된다.
-> typedef struct __employeeData {
	int cnum;
	char name[20];
} EDATA;

이를 기반으로 대략 네명의 직원 정보를 원형 연결 리스트에 저장한다.
네 명의 직원정보는 임의로 결정(scanf 안써도 됨)

원형 연결리스트에는 구조체 변수의 주소 값을 저장해야함


2. 직원은 순서대로 돌아가면서 당직을 선다.
당직의 순서는 프로그램에 등록되는 등록 순서를 기준으로 결정된다.

3. 직원의 이름과 하나의 숫자를 이용해서 당직자를 확인한다.
함수를 정의하고 직원의 이름과 숫자를 인자로 전달받는다.
전달된 이름의 직원이 당직을 서고, 전달된 숫자만큼의 날이 지나고 당직을 서게되는 직원의 정보를 반환한다.
